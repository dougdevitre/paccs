# Outreach Campaign Planning Process

A six-phase process for planning, executing, and evaluating PACCS outreach campaigns that educate decision-makers and the public about parental alienation and child-centered safety.

---

## Overview

Effective outreach requires deliberate planning, clear objectives, and rigorous evaluation. This process guides PACCS committees and volunteers through every stage of an outreach campaign — from initial concept through post-campaign analysis.

---

## Phase 1: Define Objectives

Before any campaign work begins, establish a clear foundation.

### 1.1 Target Audience

Identify the primary and secondary audiences for this campaign:

| Audience Category | Examples | Key Concerns |
|-------------------|----------|--------------|
| **Judiciary** | Family court judges, magistrates, court administrators | Case management, child welfare outcomes, judicial education credits |
| **Legislators** | State representatives, senators, committee staff | Constituent impact, fiscal notes, bipartisan appeal |
| **Legal Professionals** | Family law attorneys, GALs, mediators | CLE credits, case outcomes, client retention |
| **Mental Health Professionals** | Therapists, counselors, evaluators, psychologists | CEU credits, clinical frameworks, diagnostic tools |
| **Educators** | School counselors, teachers, administrators | Student welfare, mandatory reporting, parent communication |
| **General Public** | Parents, grandparents, community members | Awareness, recognition of warning signs, support resources |

### 1.2 Key Messages

Define 3-5 core messages for the campaign. Each message should be:

- **Clear** — understandable without specialized knowledge
- **Evidence-based** — supported by peer-reviewed research
- **Actionable** — tells the audience what they can do
- **Child-centered** — frames the issue around children's wellbeing

### 1.3 Desired Outcomes

Define specific, measurable outcomes:

- [ ] Number of materials distributed
- [ ] Number of meetings secured
- [ ] Number of professionals educated
- [ ] Media coverage generated (articles, segments, mentions)
- [ ] Policy actions initiated (bills introduced, hearings scheduled)
- [ ] Website traffic or resource downloads
- [ ] New partnerships or endorsements

### 1.4 Success Metrics

| Metric | Target | Measurement Method |
|--------|--------|--------------------|
| Materials distributed | [target number] | Distribution tracking log |
| Decision-maker meetings | [target number] | Meeting tracking log |
| Media mentions | [target number] | Media monitoring |
| Website visits | [target number] | Analytics dashboard |
| New contacts added | [target number] | CRM/contact database |
| Follow-up actions completed | [target percentage] | Follow-up tracking log |

## Phase 2: Research & Preparation

### 2.1 Audience Analysis

For each target audience, document:

- **Current awareness level** — What do they already know about parental alienation?
- **Common misconceptions** — What myths or biases might they hold?
- **Decision-making factors** — What motivates action in this audience?
- **Preferred communication channels** — How do they prefer to receive information?
- **Gatekeepers** — Who controls access to this audience?
- **Timing considerations** — Legislative sessions, conference schedules, awareness months

### 2.2 Material Inventory

Review existing PACCS materials and identify what is available and what needs to be created:

| Material | Status | Audience | Notes |
|----------|--------|----------|-------|
| Alienation Identification Checklist | Available | Parents, Evaluators | Current version |
| Legislative One-Pager | Needs update | Legislators | Update with current session data |
| Judicial Bench Card | In development | Judges | Target completion date: [date] |
| Research Summary | Available | All professionals | Latest research citations |
| Myth vs. Fact Sheet | Available | General public | Reviewed [date] |
| [Additional materials] | [Status] | [Audience] | [Notes] |

### 2.3 Resource Assessment

- **Budget** — Available funding for printing, postage, travel, events
- **Volunteers** — Number available, skills, geographic distribution
- **Timeline** — Hard deadlines (legislative sessions, awareness months, conferences)
- **Partnerships** — Allied organizations that can amplify the campaign
- **Technology** — Email platforms, social media accounts, website capabilities

---

## Phase 3: Content Development

### 3.1 Material Creation Checklist

For each new material to be created:

- [ ] Draft written by subject matter expert
- [ ] Research citations verified by Research Integrity Committee
- [ ] Lived experience review completed (Lived Experience Committee)
- [ ] Legal accuracy review completed (Legal Advisory Committee)
- [ ] Plain language review (reading level appropriate for audience)
- [ ] PACCS branding and formatting applied
- [ ] Disclaimer included on all materials
- [ ] Print-ready and digital versions prepared
- [ ] Accessibility check completed (alt text, screen reader compatible)

### 3.2 Review Process

All outreach materials must pass through the following review chain:

1. **Author/Creator** — Drafts the material
2. **Committee Chair** — Initial review for alignment with campaign objectives
3. **Research Integrity Committee** — Verifies all citations and claims
4. **Legal Advisory Committee** — Reviews for legal accuracy (if applicable)
5. **Lived Experience Committee** — Reviews for sensitivity and real-world accuracy
6. **Advisory Council** — Final approval for external distribution

### 3.3 Approval Workflow

| Stage | Reviewer | Turnaround | Action |
|-------|----------|------------|--------|
| Draft submission | Author | — | Submit to committee chair |
| Committee review | Committee Chair | 5 business days | Approve, request revisions, or escalate |
| Fact-check | Research Integrity | 5 business days | Verify citations and claims |
| Legal review | Legal Advisory | 5 business days | Confirm legal accuracy |
| Sensitivity review | Lived Experience | 5 business days | Confirm real-world accuracy |
| Final approval | Advisory Council | 3 business days | Approve for distribution |

**Total review cycle: approximately 3-4 weeks** (stages may overlap)

---

## Phase 4: Distribution

### 4.1 Distribution Channels

| Channel | Best For | Lead Time | Cost |
|---------|----------|-----------|------|
| **In-person delivery** | High-priority targets (judges, key legislators) | 1-2 weeks scheduling | Travel, printing |
| **U.S. Mail** | Formal communications, legislative packets | 1-2 weeks delivery | Printing, postage |
| **Email** | Follow-ups, digital materials, newsletters | Same day | Email platform fees |
| **Digital platforms** | Broad audience, social media, website | Same day | Content creation time |
| **Professional conferences** | Concentrated professional audiences | 3-6 months planning | Registration, booth, travel |
| **Webinars/virtual events** | National reach, recorded for reuse | 4-6 weeks planning | Platform fees |
| **Partner organizations** | Amplified reach through allied networks | 2-4 weeks coordination | Varies |

### 4.2 Distribution Timeline

| Week | Activity | Responsible |
|------|----------|-------------|
| Week 1 | Finalize all materials, confirm distribution list | Campaign Lead |
| Week 2 | Begin mail distribution, schedule in-person deliveries | Distribution Team |
| Week 3 | Email distribution, social media launch | Communications Lead |
| Week 4 | In-person deliveries, conference distributions | Field Team |
| Weeks 5-8 | Ongoing distribution, follow-up outreach | All Teams |
| Weeks 9-10 | Second wave follow-ups | Follow-Up Team |

### 4.3 Partnership Logistics

For each distribution partner:

- [ ] Partnership agreement or MOU in place
- [ ] Materials provided in partner-preferred format
- [ ] Co-branding approved (if applicable)
- [ ] Distribution timeline aligned
- [ ] Reporting expectations communicated
- [ ] Point of contact identified on both sides

---

## Phase 5: Execution

### 5.1 Launch Checklist

- [ ] All materials printed, packaged, and ready for distribution
- [ ] Distribution list finalized and segmented by channel
- [ ] Email sequences scheduled
- [ ] Social media posts scheduled
- [ ] In-person delivery schedule confirmed
- [ ] Volunteer assignments distributed
- [ ] Tracking systems active (spreadsheets, CRM, analytics)
- [ ] Media outreach initiated (press releases, media alerts)
- [ ] Partner organizations notified and materials shared
- [ ] Backup plans documented for common issues

### 5.2 Monitoring

During campaign execution, track:

- **Daily** — Social media engagement, email open/click rates, website traffic
- **Weekly** — Materials distributed, meetings completed, media coverage
- **Biweekly** — Campaign team check-in meeting to review progress
- **As needed** — Issue resolution, volunteer support, material replenishment

### 5.3 Real-Time Adjustments

Be prepared to adjust the campaign based on:

- **Positive signals** — High engagement on specific content (amplify it)
- **Low engagement** — Underperforming channels or messages (diagnose and adjust)
- **External events** — News stories, legislative actions, or public events that create opportunities
- **Feedback** — Direct responses from target audiences that suggest message refinement
- **Resource constraints** — Volunteer availability, budget adjustments

---

## Phase 6: Evaluation

### 6.1 Metrics Review

Within 2 weeks of campaign conclusion, compile:

| Metric | Target | Actual | Variance | Notes |
|--------|--------|--------|----------|-------|
| Materials distributed | | | | |
| Decision-maker meetings | | | | |
| Media mentions | | | | |
| Website visits | | | | |
| New contacts | | | | |
| Follow-up completion rate | | | | |
| [Custom metric] | | | | |

### 6.2 Feedback Collection

- **Internal feedback** — Debrief with all campaign team members and volunteers
- **External feedback** — Survey or interview recipients and partners
- **Audience response** — Analyze replies, questions, and actions taken by targets
- **Media analysis** — Review coverage for accuracy, tone, and reach

### 6.3 Lessons Learned

Document answers to:

1. What worked well and should be repeated?
2. What did not work and should be changed?
3. What was missing that should be added next time?
4. Were the right audiences targeted?
5. Were the messages effective?
6. Were resources (budget, time, volunteers) sufficient?
7. What unexpected challenges arose and how were they handled?
8. What partnerships were most valuable?

### 6.4 Reporting

Prepare a post-campaign report for PACCS leadership that includes:

- Executive summary
- Campaign objectives vs. outcomes
- Metrics dashboard
- Key wins and highlights
- Challenges and how they were addressed
- Lessons learned
- Recommendations for future campaigns
- Budget summary (planned vs. actual)
- Appendix: sample materials, media coverage, notable responses
