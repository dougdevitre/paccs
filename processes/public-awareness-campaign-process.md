# Public Awareness Campaign Process

> **Audience:** PACCS committees, advocates, campaign organizers
> **Purpose:** Plan and execute public awareness campaigns for child-centered safety
> **Related Skill:** paccs-tech-policy-education

---

## Campaign Types

| Type | Trigger | Duration | Example |
|------|---------|----------|---------|
| **Awareness Month** | Annual calendar event | 4 weeks | Parental Alienation Awareness Month (April) |
| **Legislative Response** | Bill introduced or approaching vote | 2-4 weeks | Opposition to harmful custody bill |
| **Research Release** | New study or data published | 1-2 weeks | Announcing new alienation outcomes research |
| **Crisis Response** | Harmful media coverage or policy change | 1-5 days | Rapid response to misinformation |
| **Community Event** | Conference, walk, vigil, webinar | 4-8 weeks prep | Annual PACCS awareness event |

## 12-Week Campaign Planning Timeline

### Weeks 1-2: Strategy & Planning

- [ ] Define campaign objective (one clear, measurable goal)
- [ ] Identify target audience (parents, public, professionals, legislators)
- [ ] Develop key messages (3-5 core messages aligned with Five Pillars)
- [ ] Set success metrics (reach, engagement, media mentions, policy impact)
- [ ] Establish budget (even if $0 — identify what you need)
- [ ] Assign team roles (lead, content creator, social media, media liaison, logistics)

### Weeks 3-4: Content Development

- [ ] Create or adapt materials from PACCS library:
  - [ ] Social media posts (from `outreach/social-media/social-media-campaign-kit.md`)
  - [ ] Press releases (from `outreach/media/press-release-template.md`)
  - [ ] Fact sheets or myth-vs-fact sheets (from `outreach/myth-vs-fact/`)
  - [ ] Video scripts (from `outreach/video-scripts/`)
- [ ] Design visual assets (graphics, infographics, banners)
- [ ] Draft email/newsletter content
- [ ] Prepare media kit (press release, fact sheet, spokesperson bio, key quotes)
- [ ] Review all content for PACCS guardrails compliance

### Weeks 5-6: Partnership & Media Outreach

- [ ] Identify allied organizations for co-promotion
- [ ] Reach out to partners with co-branded opportunities
- [ ] Build media list (local newspapers, radio, TV, podcasts, online outlets)
- [ ] Send media advisory or pitch to journalists
- [ ] Identify and brief spokespeople (see `resources/scripts/media-interview/media-talking-points.md`)
- [ ] Schedule any events (webinars, rallies, press conferences)

### Weeks 7-8: Pre-Launch

- [ ] Schedule all social media posts
- [ ] Send pre-launch emails to supporters and partners
- [ ] Confirm all event logistics
- [ ] Brief all team members on roles and messaging
- [ ] Prepare rapid-response protocol for negative reactions
- [ ] Test all links, sign-up forms, and digital tools

### Weeks 9-10: Launch & Execution

- [ ] Launch campaign on designated date
- [ ] Publish social media content per calendar
- [ ] Distribute press release
- [ ] Host events as scheduled
- [ ] Monitor social media and respond to engagement
- [ ] Track metrics daily
- [ ] Adjust messaging based on what's resonating

### Weeks 11-12: Sustain & Wrap Up

- [ ] Continue content distribution
- [ ] Send follow-up media pitches for coverage
- [ ] Share user-generated content and testimonials
- [ ] Host closing event or summary webinar
- [ ] Send thank-you messages to partners and supporters
- [ ] Begin evaluation

## Event Planning Checklist

### Webinar / Virtual Event
- [ ] Select platform (Zoom, Teams, YouTube Live)
- [ ] Book speakers/panelists
- [ ] Create registration page
- [ ] Promote through all channels (2-3 weeks lead time)
- [ ] Prepare presentation materials
- [ ] Test technology
- [ ] Record the event for later distribution
- [ ] Follow up with attendees (recording link, resources, next steps)

### In-Person Event (Rally, Walk, Conference)
- [ ] Secure venue and permits
- [ ] Arrange speakers and schedule
- [ ] Create printed materials (signs, handouts, banners)
- [ ] Coordinate media coverage
- [ ] Arrange photography/videography
- [ ] Plan logistics (parking, accessibility, weather contingency)
- [ ] Prepare safety protocol
- [ ] Follow up with attendees and media

## Handling Negative Reactions

### Common Attacks & Responses

| Attack | Response Strategy |
|--------|------------------|
| "Parental alienation is debunked" | Share myth-vs-fact sheet; cite peer-reviewed research |
| "This protects abusers" | Emphasize safety screening; cite Five Pillars (Protection from System Abuse) |
| "Children should choose" | Explain independent thinker phenomenon; cite developmental research |
| Online harassment or trolling | Do not engage; document; block if necessary; report threats |
| Media mischaracterization | Issue correction with evidence; offer spokesperson for interview |

### Crisis Response Protocol
1. **Assess** — Is this a one-off comment or a coordinated campaign?
2. **Document** — Screenshot and save all relevant content
3. **Consult** — Brief PACCS leadership before responding publicly
4. **Respond** — Use prepared messaging; stay evidence-based and calm
5. **Monitor** — Track for escalation over 24-48 hours
6. **Report** — If threats are made, report to platform and law enforcement
