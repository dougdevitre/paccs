# PACCS Communication Scripts — Index

Guided conversation scripts for specific interactions encountered by targeted parents, advocates, and PACCS representatives. Each script provides word-for-word or structured language for real-world situations, following PACCS guardrails: no professional advice, safety first, evidence integrity, child-centered, privacy protection, and both-parents-matter.

> **Usage:** These scripts are educational tools, not professional advice. Every situation is unique. Always work with qualified professionals (attorneys, therapists, evaluators) for case-specific guidance.

## PACCS Guardrails Applied to All Scripts

All scripts in this collection adhere to the following principles:

- **No professional advice.** Scripts are educational. They do not replace attorneys, therapists, or other qualified professionals.
- **Safety first.** If any situation involves a threat to physical safety, the guidance is to contact law enforcement immediately.
- **Evidence integrity.** All suggested language is designed to be court-admissible — factual, timestamped, and free of emotional or inflammatory language.
- **Child-centered.** Every script prioritizes the child's wellbeing, safety, and right to a relationship with both parents.
- **Privacy protection.** No script encourages sharing identifying information about children, co-parents, or ongoing cases.
- **Both-parents-matter.** Scripts never position one parent as inherently superior or encourage language that demonizes the other parent.
